Le regole organizzative ("nessun bucket S3 pubblico", "tutti i volumi cifrati", "no security group `0.0.0.0/0` su porte non standard") espresse come codice eseguibile, non come PDF. **OPA/Conftest, HashiCorp Sentinel, Checkov, tfsec, Terrascan**. Dove entrano in pipeline (pre-plan, post-plan, in CI). **Drift detection**: come accorgersi che la console è stata toccata a mano. **GitOps** come pattern di chiusura del cerchio: il repository Git è la fonte di verità, divergere è un alert.
